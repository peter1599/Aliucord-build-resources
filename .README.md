# Aliucord-build-resources
A temporary solution to compile resources for Aliucord plugins. (Windows users only)

# Installation
1. Download and place all the files expect README and start-build.au3(start-build.au3 is the source code of the build-resources.exe) in the main folder of your project direcory. (This is where you found the gradlew.bat )
2. After your plugin build is done it will automatically start the build-resources.exe
